Detailed Description

This class provides network traffic statistics.

See also
PhotonPeer::getTrafficStatsEnabled(), PhotonPeer::setTrafficStatsEnabled(), PhotonPeer::getTrafficStatsElapsedMs(), PhotonPeer::getTrafficStatsIncoming(), PhotonPeer::getTrafficStatsOutgoing()
